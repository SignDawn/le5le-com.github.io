# 定义

## 类型

### GraphType
操作器类型
| 可选值       | 类型   | 说明       |
| ------------ | ------ | ---------- |
| `'position'` | string | 位置操作器 |
| `'rotation'` | string | 旋转操作器 |
| `'scaling'`  | string | 缩放操作器 |
### GridUnit
场景单位
| 可选值         | 类型   | 说明 |
| -------------- | ------ | ---- |
| `'centimeter'` | string | 厘米 |
| `'decimeter'`  | string | 分米 |
| `'meter'`      | string | 米   |
